# Kwork — скрытие проектов

Chrome-расширение добавляет кнопку «Скрыть» на карточки страницы [kwork.ru/projects](https://kwork.ru/projects). Скрытые проекты сохраняются локально и не возвращаются после обновления страницы или перезапуска браузера. Последнее действие можно отменить в течение шести секунд.

Штатная пагинация заменяется единой лентой: при приближении к её концу следующая страница Kwork автоматически подгружается и добавляется ниже уже показанных проектов. Скрытые проекты не создают пустые или разреженные страницы.

## Установка

1. Откройте в Chrome страницу `chrome://extensions`.
2. Включите «Режим разработчика» в правом верхнем углу.
3. Нажмите «Загрузить распакованное расширение».
4. Выберите папку `kwork-project-hider`.
5. Откройте или обновите `https://kwork.ru/projects`.

Попапа и страницы настроек нет. Расширение запрашивает только доступ к локальному хранилищу Chrome и странице биржи проектов Kwork.

## Использование

Нажмите «Скрыть» внизу карточки проекта. Карточка исчезнет, а в нижней части окна появится действие «Отменить». Если отмену не нажать, проект останется скрытым во всех будущих сессиях этого профиля Chrome.

Для просмотра следующих проектов просто прокручивайте страницу вниз. Номера страниц больше не показываются; во время запроса появляется короткий статус загрузки, а в конце — сообщение, что все доступные проекты загружены.
